[[GitHub]]   [[IntelliJ]]

 *PROCEDURES* :
 
 1. Open IntelliJ --> git(top) -->fetch --> cross check in master option (Top) 
 2.  Hover on your branch name --> checkout 
 3.  You can open GitBash and check branch name changed to cross check
 4.  Branch name at the top should be changed to your branch name in IntelliJ
 5.  Create your program 
 6.  LOCAL TO REMOTE:
      Top left (commit option symbol)--> click --> select which files to export --> enter commit message --> Commit -->  Now file comes inside changes tab --> git --> Push --> files added will be displayed --> select PUSH (not force push) 
 7. Crosscheck your push in GitHub
